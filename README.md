内存优化易语言模块（项目已停更）


1

该模块是属于一个程序内存优化，会对其使用程序的内存占用进行清理，比如打开程序后内存占用为50M，在相关程序代码加入该模块，可以对程序进行内存占用优化清理，可以从50M减低为5M以下，不过具体要看该程序运行情况，如果出现问题那就是对其模块不兼容。
